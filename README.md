# 底辺テトリス 〜麻呂&織部ブロック〜

🎮 **高品質な名言入りテトリスゲーム**

各ブロックにはユニークな日本語の名言が入っています！
スマホ縦画面に最適化された操作で、いつでもどこでも楽しめます。

## 🕹️ プレイ方法

**GitHub Pages:** [https://yurinko-teihen.github.io/teihen_Tetris/](https://yurinko-teihen.github.io/teihen_Tetris/)

### 操作方法

#### タッチ操作（スマホ）
- **左右ボタン**: ブロックを左右に移動
- **回転ボタン (🔄)**: ブロックを回転
- **下ボタン (▼)**: ブロックを下に移動
- **ドロップボタン (⏬)**: 即座に落下

#### キーボード操作（PC）
- **←/→**: 左右移動
- **↓**: 下に移動
- **↑/スペース**: 回転
- **Enter**: ハードドロップ
- **Escape/P**: 一時停止

## 🎯 ゲームルール

- **10ライン消すとレベルアップ**
- **レベル10でゲームクリア**
- レベルが上がると落下速度がアップ
- 同時に複数ライン消すと高得点！

## 🔧 カスタマイズ

### ブロックの追加方法

`blocks.js` の `BLOCK_DEFINITIONS` 配列に新しいブロックを追加できます：

```javascript
{
    id: 'NEW1',           // ユニークなID
    shape: [              // ブロックの形（1がブロック、0が空白）
        [1, 1],
        [1, 1]
    ],
    chars: [              // 各セルに表示する文字
        ['あ', 'い'],
        ['う', 'え']
    ],
    quote: '「新しい名言」',  // 画面下部に表示される名言
    color: '#ff6b6b',       // メインカラー
    shadowColor: '#ee5a5a', // 影の色
    gradient: ['#ff6b6b', '#ee5a5a']  // グラデーション
}
```
